# 深度学习作业二 · 函数拟合实验报告

## 一、函数定义

本实验选取的目标函数为：

$$y = f(x) = \sin(x) + 0.3 \cos(2x), \quad x \in [-\pi, \pi]$$

该函数在有限区间 $[-\pi, \pi]$ 上连续，由正弦与余弦叠加而成，具有一定非线性，适合用于验证两层 ReLU 神经网络的函数逼近能力。理论表明，具有单隐藏层和足够神经元的 ReLU 网络可以以任意精度逼近此类连续函数。

---

## 二、数据采集

- **定义域**：$x \in [-\pi, \pi]$。
- **训练集**：在定义域内均匀采样 **500** 个点，对每个点的函数值加入均值为 0、标准差为 0.05 的高斯噪声，以模拟实际数据中的测量误差。
- **测试集**：在相同区间内均匀采样 **200** 个点，不添加噪声，用于客观评估模型的泛化与拟合效果。
- **随机种子**：固定 `np.random.seed(42)`，保证实验可复现。

---

## 三、模型描述

- **网络结构**：两层全连接（一层隐藏层 + 一层输出层）
  - 输入层：1 维（标量 $x$）
  - 隐藏层：64 个神经元，激活函数为 **ReLU**
  - 输出层：1 维（回归值 $\hat{y}$）
  - 总参数量：193（含偏置）
- **损失函数**：均方误差（MSE），即 $\mathcal{L} = \frac{1}{n}\sum_{i}(y_i - \hat{y}_i)^2$。
- **优化器**：Adam，学习率 0.01。
- **训练设置**：200 个 epoch，batch size 32；使用测试集作为验证集监控泛化表现。
- **实现框架**：TensorFlow 2 / Keras。

---

## 四、拟合效果

- **训练集**：MSE 约 0.003，MAE 约 0.044，说明模型在带噪声的训练数据上能够有效拟合。
- **测试集**：MSE 约 0.001，MAE 约 0.022；测试误差低于训练误差，符合“测试集无噪声、训练集有噪声”的设定，表明模型泛化良好、未明显过拟合。
- **可视化**：真实曲线 $f(x)$ 与网络预测曲线在 $[-\pi, \pi]$ 上高度重合，仅在少数波动较大或噪声较大的区域存在轻微偏差，整体验证了两层 ReLU 网络对该函数的拟合能力。

**结论**：在给定数据与超参数下，两层 ReLU 网络能够较好地逼近 $f(x) = \sin(x) + 0.3\cos(2x)$，与万能近似定理的结论一致。完整代码与图表见 `function_fitting.ipynb`。
